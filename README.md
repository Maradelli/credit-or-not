# Credit card approval model
Целью данного проекта является оценивание платёжеспособности клиента банка с помощью модели машинного обучения для решения вопроса о выдаче кредитной карты.

## Технологии (стек):
  - **Python**: основной инструмент для визуализации (*Seaborn, Matplotlib*), обработки данных (*Pandas*), вычислений (*Numpy*) и машинного обучения (*scikit-learn, XGBoost*)
  - **Docker**: контейнеризация готового приложения
  - **MS Excel**: загрузка и выгрузка данных в формате .xlsx в процессе работы приложения
